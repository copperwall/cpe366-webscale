Assumptions:

1. Weeks start with Monday and end with Sunday
2. Two-week periods start with the first and third full weeks of the month
3. An admin will not request shifts nor vacation/sickdays
4. An Employee will not want to change their own password
5. No two users will be using the application at the same time
6. No invalid login/password will be entered when creating a new employee
7. Sick days will not be requested more than 3 weeks away.
